# Simple Rubik's Cube

使用 Three.js 编写的极简 3D 魔方

三阶还原使用了 [cubejs](https://github.com/ldez/cubejs) 算法库

旋转方向的判断参考了 https://github.com/pancerZH/I-cannot-deal-with-Rubiks-cube 大佬的实现

# 功能

- 随机旋转
- 还原魔方（仅限三阶）
- 支持任意阶
  - 在源码中修改全局变量 N

# 使用

下载项目，双击打开 index.html 文件
